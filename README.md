# customer-crud-pandas
Simple CRUD client management system (create, read, update, delete) in Python using Pandas Library and CSV for data storage.

## Features
- Register new clients
- List all clients
- Search clients by CPF
- Update client information
- Delete clients
- Data persistence using CSV

## Technologies
- Python
- Pandas Library
- CSV file storage
